# Zara Sales EDA

## Overview
Exploratory data analysis of a fashion retail dataset focusing on seasonality,
pricing structure, and material composition.

## Key Insights
- Strong seasonality with peak sales in Autumn
- Jackets dominate sales across all seasons (unexpected pattern)
- Uniform median pricing across categories
- Seasonal shift in material composition

## Business Recommendations
- Revisit pricing strategy by category
- Investigate quality and suitability of summer materials

## Tools
- Power BI
- Kaggle dataset
